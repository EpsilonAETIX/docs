Epsilon HUD - Yapilandirma

Tum ayarlar config.lua dosyasindan yapilir. Asagida her ayarin detayli aciklamasi bulunmaktadir.

Genel Ayarlar

config.lua 
Config = {}

Config.Framework = 'esx'         -- 'esx' veya 'qbcore'
Config.DefaultTheme = 'dark'     -- 'dark' veya 'light'
Config.SpeedUnit = 'kmh'         -- 'kmh' veya 'mph'
Config.MinimapShape = 'circle'   -- 'circle' veya 'square'
Config.ShowStreetName = true     -- Sokak adi gosterimi
Config.ShowCompass = true        -- Pusula gosterimi

Ayar Tablosu

AyarTipVarsayilanAciklama
Config.Frameworkstring'esx'Kullanilacak framework
Config.DefaultThemestring'dark'Varsayilan tema
Config.SpeedUnitstring'kmh'Hiz birimi
Config.MinimapShapestring'circle'Minimap sekli
Config.ShowStreetNamebooleantrueSokak adi goster/gizle
Config.ShowCompassbooleantruePusula goster/gizle

Bar Ayarlari

Her barin gorunurlugu ve rengini ayarlayabilirsiniz:

config.lua 
Config.Bars = {
    health = { show = true, color = '#3fb950' },
    armor  = { show = true, color = '#58a6ff' },
    hunger = { show = true, color = '#d29922' },
    thirst = { show = true, color = '#58a6ff' },
    stress = { show = true, color = '#f85149' },
    oxygen = { show = true, color = '#79c0ff' },
}

Arac HUD Ayarlari

config.lua 
Config.VehicleHUD = {
    showSpeed = true,
    showFuel = true,
    showDamage = true,
    showSeatbelt = true,
    fuelScript = 'LegacyFuel'   -- Yakit scripti adi
}
Dikkat: fuelScript degerini kullandiginiz yakit scriptinin resource adina gore degistirin. Ornegin ox_fuel, LegacyFuel vb.
