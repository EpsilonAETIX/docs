Epsilon Inventory - Item Yonetimi

Envantere yeni item ekleme, gorsel ayarlari ve meta data kullanimi.

Yeni Item Ekleme

items.lua dosyasina yeni item tanimlayin:

items.lua 
Items = {
    ['bread'] = {
        label = 'Ekmek',
        weight = 200,
        stackable = true,
        description = 'Taze ekmek, acligi giderir.',
        usable = true,
        image = 'bread.png'
    },
    ['lockpick'] = {
        label = 'Maymuncuk',
        weight = 300,
        stackable = true,
        description = 'Arac kapisini acmak icin kullanilir.',
        usable = true,
        image = 'lockpick.png'
    },
    ['radio'] = {
        label = 'Telsiz',
        weight = 500,
        stackable = false,
        description = 'Iletisim icin kullanilir.',
        usable = true,
        image = 'radio.png',
        unique = true
    }
}

Item Ozellikleri

OzellikTipZorunluAciklama
labelstringEvetItemin gorunen adi
weightnumberEvetAgirlik (gram)
stackablebooleanEvetUst uste yigilabilir mi
descriptionstringHayirItem aciklamasi
usablebooleanHayirKullanilabilir mi
imagestringHayirGorsel dosya adi
uniquebooleanHayirBenzersiz mi (meta data icin)

Item Gorselleri

Item gorselleri html/images/ klasorune PNG formatinda eklenir.

  • Onerilen boyut: 128x128px
  • Format: PNG (seffaf arka plan)
  • Dosya adi, item adinin aynisi olmali (orn: bread.png)

Meta Data Kullanimi

Benzersiz itemlere ek bilgi eklemek icin meta data kullanilir:

Server - Lua 
-- Meta data ile item ekle
exports['epsilon-inventory']:addItem(source, 'weapon_pistol', 1, {
    serial = 'EPS-' .. math.random(100000, 999999),
    durability = 100,
    ammo = 30
})

-- Meta data oku
local items = exports['epsilon-inventory']:getItems(source)
for _, item in pairs(items) do
    if item.name == 'weapon_pistol' then
        print('Seri No:', item.metadata.serial)
        print('Dayaniklilik:', item.metadata.durability)
    end
end

Kullanilabilir Item Kaydi

Bir itemi kullanilabilir yapmak icin server tarafinda kayit edin:

Server - Lua 
exports['epsilon-inventory']:registerUsableItem('bread', function(source, item)
    local xPlayer = ESX.GetPlayerFromId(source)

    -- Itemi cikar
    exports['epsilon-inventory']:removeItem(source, 'bread', 1)

    -- Aclik degerini artir
    TriggerClientEvent('esx_status:add', source, 'hunger', 200000)

    -- Bildirim
    xPlayer.showNotification('Ekmek yediniz.')
end)
