# Epsilon Scripts - FiveM Dökümanlar

FiveM sunucuları için gelişmiş, optimize edilmiş ve kolay kullanılabilir scriptlerin döküman sitesi.

## 🌐 Canlı Site

[https://epsilon-scripts.github.io/docs](https://epsilon-scripts.github.io/docs)

## 📂 Yapı

```
epsilon/
├── index.html              # Ana sayfa
├── assets/
│   ├── css/style.css       # Stiller
│   └── js/main.js          # JavaScript
├── pages/
│   ├── kurulum.html        # Kurulum rehberi
│   ├── genel-ayarlar.html  # Genel ayarlar
│   ├── faq.html            # Sıkça Sorulan Sorular
│   ├── changelog.html      # Değişiklik günlüğü
│   ├── scripts/            # Script dökümanları
│   │   ├── epsilon-hud.html
│   │   ├── epsilon-inventory.html
│   │   ├── epsilon-phone.html
│   │   ├── epsilon-garage.html
│   │   ├── epsilon-housing.html
│   │   └── epsilon-job.html
│   └── api/                # API dökümanları
│       ├── exports.html
│       ├── events.html
│       └── callbacks.html
└── _config.yml             # GitHub Pages config
```

## 🚀 Yeni Script Dökümanı Ekleme

1. `pages/scripts/` klasörüne yeni bir HTML dosyası oluşturun
2. Mevcut bir script sayfasını şablon olarak kullanın
3. `index.html` ve tüm sidebar navigasyonlarına yeni linki ekleyin
4. Commit & push yapın
