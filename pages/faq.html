Frequently Asked Questions

Answers to the most commonly asked questions.

General

Which frameworks do you support?

All of our scripts support ESX Legacy and QBCore frameworks. Some scripts can also run in standalone mode.

Are the scripts free?

Some of our scripts are completely free and open source. For premium scripts, please visit our Discord server.

Is OneSync required?

Yes, all of our scripts require OneSync (Legacy or Infinity).

Troubleshooting

The script throws an error on startup

Check the error message in the F8 console. The most common issues are:

  • Missing dependency – make sure all required scripts are installed
  • Wrong framework setting – check the framework value in config.lua
  • Missing SQL tables – make sure you have imported the SQL files

NUI (interface) is not opening

  • Clear the browser cache: type nui_devtools in the F8 console
  • Make sure the UI files listed in fxmanifest.lua are correct

I am getting a database error

  • Make sure the oxmysql script is installed
  • Check the mysql_connection_string value in server.cfg
  • Make sure the SQL tables have been created
If you couldn't find a solution, you can get support by joining our Discord server.
