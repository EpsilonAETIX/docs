Epsilon Inventory - Kurulum

Envanter scriptini sunucunuza kurmak icin asagidaki adimlari takip edin.

Adim 1: SQL Import

Asagidaki SQL dosyasini veritabaniniza import edin:

install.sql 
CREATE TABLE IF NOT EXISTS `epsilon_inventory` (
    `id` INT AUTO_INCREMENT PRIMARY KEY,
    `owner` VARCHAR(60) NOT NULL,
    `name` VARCHAR(50) NOT NULL,
    `type` VARCHAR(20) DEFAULT 'player',
    `slots` INT DEFAULT 50,
    `weight` INT DEFAULT 120000,
    `items` LONGTEXT,
    UNIQUE KEY `owner_name` (`owner`, `name`)
);

CREATE TABLE IF NOT EXISTS `epsilon_inventory_stash` (
    `id` INT AUTO_INCREMENT PRIMARY KEY,
    `stash_id` VARCHAR(100) NOT NULL UNIQUE,
    `items` LONGTEXT,
    `created_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
Dikkat: SQL dosyasini import etmeden once mevcut veritabaninizin yedegini almayi unutmayin.

Adim 2: Dosyalari Yerlestirme

  1. Scripti indirin ve resources/[epsilon]/epsilon-inventory klasorune koyun
  2. config.lua dosyasini duzenleyin (Yapilandirma sayfasina bakin)

Adim 3: server.cfg

server.cfg 
# Bagimliliklar (once bunlar yuklenmelidir)
ensure oxmysql
ensure ox_lib
ensure es_extended  # veya qb-core

# Envanter
ensure epsilon-inventory
Onemli: Eger baska bir envanter scripti kullaniyorsaniz (ox_inventory, qs-inventory vb.), onu server.cfg'den kaldirin veya yorum satirina alin.
