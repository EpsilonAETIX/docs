⚙️ Genel Ayarlar

Tüm Epsilon scriptlerinde ortak olan genel ayarlar ve yapılandırma seçenekleri.

Framework Seçimi

Tüm Epsilon scriptleri ESX Legacy ve QBCore framework'lerini destekler. Framework ayarı her scriptin config.lua dosyasında yapılır:

config.lua 
-- ESX için
Config.Framework = 'esx'

-- QBCore için
Config.Framework = 'qbcore'

Dil Ayarları

Scriptler çoklu dil desteğine sahiptir. Dil dosyaları locales/ klasöründe bulunur:

locales/tr.lua 
Locales['tr'] = {
    ['inventory_full'] = 'Envanteriniz dolu!',
    ['not_enough_money'] = 'Yeterli paranız yok!',
    ['item_used'] = '%s kullanıldı.',
    ['item_received'] = '%s x%d alındı.',
}

Debug Modu

Sorun giderme için debug modunu aktifleştirebilirsiniz:

config.lua 
Config.Debug = true  -- Konsola detaylı log yazdırır
Dikkat: Debug modunu production sunucunuzda kapalı tutun. Performans kaybına neden olabilir.

Veritabanı Ayarları

Tüm scriptler oxmysql kullanır. server.cfg dosyanızda bağlantı stringinin doğru olduğundan emin olun:

server.cfg 
set mysql_connection_string "mysql://root:password@localhost/fivem?charset=utf8mb4"
