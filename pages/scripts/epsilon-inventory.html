Ana Sayfa Scriptler Epsilon Inventory

📦 Epsilon Inventory

Sürükle-bırak destekli, modern arayüzlü gelişmiş envanter sistemi. Crafting, hotbar, silah eklentileri ve daha fazlası.

Versiyon: 3.0.0  |  Resmon: 0.02ms  |  Framework: ESX / QBCore  |  Fiyat: Premium

Özellikler

  • ✅ Sürükle-bırak item taşıma
  • ✅ Ağırlık tabanlı envanter sistemi
  • ✅ Hotbar (1-5 tuşları ile hızlı erişim)
  • ✅ Silah eklentileri (silencer, scope, grip vb.)
  • ✅ Crafting sistemi
  • ✅ Mağaza, araç bagajı, stash entegrasyonu
  • ✅ Item meta data desteği (seri no, durability)
  • ✅ Özelleştirilebilir item görselleri
  • ✅ Context menü (sağ tık)
  • ✅ Yere item bırakma (3D props)

Bağımlılıklar

Kaynak Zorunlu Açıklama
ESX Legacy veya QBCore ✅ Evet Framework
oxmysql ✅ Evet Veritabanı
ox_lib ✅ Evet UI kütüphanesi

Kurulum

1. SQL Dosyasını Import Edin

install.sql 
CREATE TABLE IF NOT EXISTS `epsilon_inventory` (
    `id` INT AUTO_INCREMENT PRIMARY KEY,
    `owner` VARCHAR(60) NOT NULL,
    `name` VARCHAR(50) NOT NULL,
    `type` VARCHAR(20) DEFAULT 'player',
    `slots` INT DEFAULT 50,
    `weight` INT DEFAULT 120000,
    `items` LONGTEXT,
    UNIQUE KEY `owner_name` (`owner`, `name`)
);

CREATE TABLE IF NOT EXISTS `epsilon_inventory_stash` (
    `id` INT AUTO_INCREMENT PRIMARY KEY,
    `stash_id` VARCHAR(100) NOT NULL UNIQUE,
    `items` LONGTEXT,
    `created_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

2. server.cfg'ye Ekleyin

server.cfg 
ensure oxmysql
ensure ox_lib
ensure es_extended

ensure epsilon-inventory

Yapılandırma

config.lua 
Config = {}

Config.Framework = 'esx'
Config.Language = 'tr'

-- Envanter Ayarları
Config.MaxSlots = 50           -- Maksimum slot sayısı
Config.MaxWeight = 120000      -- Maksimum ağırlık (gram)
Config.HotbarSlots = 5         -- Hotbar slot sayısı

-- Özellikler
Config.EnableCrafting = true   -- Crafting sistemi
Config.EnableHotbar = true     -- Hotbar sistemi
Config.EnableDrop = true       -- Yere bırakma
Config.DropDespawnTime = 300   -- Yerdeki item silinme süresi (saniye)

-- Silah Eklentileri
Config.WeaponAttachments = true
Config.WeaponDurability = true -- Silah dayanıklılık sistemi

-- Mağaza Ayarları
Config.Shops = {
    ['market_1'] = {
        label = '24/7 Market',
        coords = vector3(25.7, -1347.3, 29.5),
        items = {
            { name = 'bread',  price = 5,  label = 'Ekmek' },
            { name = 'water',  price = 3,  label = 'Su' },
            { name = 'burger', price = 10, label = 'Hamburger' },
        }
    }
}

Exports

Envanter Aç/Kapat

Client - Lua 
-- Oyuncu envanterini aç
exports['epsilon-inventory']:openInventory()

-- Belirli bir stash'ı aç
exports['epsilon-inventory']:openStash('police_evidence', {
    label = 'Polis Delil Deposu',
    slots = 100,
    weight = 500000
})

-- Araç bagajını aç
exports['epsilon-inventory']:openTrunk(plate)

Item İşlemleri (Server)

Server - Lua 
-- Item ekle
exports['epsilon-inventory']:addItem(source, 'bread', 5)

-- Item çıkar
exports['epsilon-inventory']:removeItem(source, 'bread', 2)

-- Item kontrol
local hasItem = exports['epsilon-inventory']:hasItem(source, 'bread', 1)

-- Tüm itemleri getir
local items = exports['epsilon-inventory']:getItems(source)

-- Item sayısını getir
local count = exports['epsilon-inventory']:getItemCount(source, 'bread')

Yeni Item Ekleme

Yeni item eklemek için items.lua dosyasını düzenleyin:

items.lua 
Items = {
    ['bread'] = {
        label = 'Ekmek',
        weight = 200,
        stackable = true,
        description = 'Taze ekmek, açlığı giderir.',
        usable = true,
        image = 'bread.png'  -- html/images/ klasöründe olmalı
    },
    ['lockpick'] = {
        label = 'Maymuncuk',
        weight = 300,
        stackable = true,
        description = 'Araç kapısını açmak için kullanılır.',
        usable = true,
        image = 'lockpick.png'
    }
}
Item görselleri html/images/ klasörüne PNG formatında eklenmeli. Önerilen boyut: 128x128px.
