Ana Sayfa Kurulum Rehberi

Kurulum Rehberi

Epsilon scriptlerini FiveM sunucunuza nasıl kuracağınızı adım adım öğrenin.

Gereksinimler

Scriptleri kullanabilmek için aşağıdaki gereksinimlerin karşılanması gerekmektedir:

  • FiveM Server - Son sürüm artifact (6000+)
  • Framework - ESX Legacy veya QBCore
  • oxmysql - Veritabanı bağlantısı için
  • ox_lib - Genel kütüphane (bazı scriptler için)
Her scriptin kendine özel bağımlılıkları olabilir. Lütfen ilgili scriptin dökümanına göz atın.

İndirme

Scriptleri GitHub üzerinden indirebilirsiniz:

Git ile İndirme

Terminal 
cd /sunucu-klasoru/resources/[epsilon]
git clone https://github.com/epsilon-scripts/epsilon-hud.git

Manuel İndirme

  1. İlgili scriptin GitHub sayfasına gidin
  2. Code → Download ZIP butonuna tıklayın
  3. ZIP dosyasını resources/[epsilon] klasörüne çıkarın
  4. Klasör adındaki -main veya -master ekini silin

Veritabanı Kurulumu

Her scriptin kendi klasöründe bir sql dosyası bulunur. Bu dosyaları veritabanınıza import edin:

SQL 
-- HeidiSQL veya phpMyAdmin üzerinden import edin:
-- resources/[epsilon]/epsilon-hud/sql/install.sql
-- resources/[epsilon]/epsilon-inventory/sql/install.sql
-- ...
Dikkat: SQL dosyalarını import etmeden önce mevcut veritabanınızın yedeğini almayı unutmayın!

Yapılandırma

Her scriptin kendi config.lua dosyası vardır. Scripti başlatmadan önce yapılandırmanızı tamamlayın:

config.lua 
Config = {}

Config.Framework = 'esx' -- 'esx' veya 'qbcore'
Config.Language = 'tr'    -- Dil seçimi
Config.Debug = false      -- Debug modu

-- Diğer ayarlar scripte göre değişir
-- Her scriptin dökümanında detaylı açıklamalar mevcuttur

Scripti Başlatma

server.cfg dosyanıza aşağıdaki satırları ekleyin:

server.cfg 
# Bağımlılıklar (önce bunlar yüklenmeli)
ensure oxmysql
ensure ox_lib
ensure es_extended  # veya qb-core

# Epsilon Scriptler
ensure epsilon-hud
ensure epsilon-inventory
ensure epsilon-phone
