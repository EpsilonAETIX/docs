General Settings

Common general settings and configuration options shared across all Aetech scripts.

Framework Selection

config.lua
Config.Framework = 'esx'     -- 'esx' veya 'qbcore'

Language Settings

Scripts support multiple languages. Language files are located in the locales/ folder:

locales/tr.lua
Locales['tr'] = {
    ['inventory_full'] = 'Envanteriniz dolu!',
    ['not_enough_money'] = 'Yeterli paraniz yok!',
    ['item_used'] = '%s kullanildi.',
    ['item_received'] = '%s x%d alindi.',
}

Debug Mode

config.lua
Config.Debug = true  -- Prints detailed logs to the console
Warning: Keep debug mode disabled on your production server. It may cause performance issues.

Database Settings

server.cfg
set mysql_connection_string "mysql://root:password@localhost/fivem?charset=utf8mb4"
