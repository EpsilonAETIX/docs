Ana Sayfa Kurulum Rehberi

Kurulum Rehberi

Epsilon scriptlerini FiveM sunucunuza nasil kuracaginizi adim adim ogrenin.

Gereksinimler

  • FiveM Server - Son surum artifact (6000+)
  • Framework - ESX Legacy veya QBCore
  • oxmysql - Veritabani baglantisi icin
  • ox_lib - Genel kutuphane (bazi scriptler icin)
Her scriptin kendine ozel bagimliliklari olabilir. Lutfen ilgili scriptin dokumanina goz atin.

Indirme

Git ile Indirme

Terminal
cd /sunucu-klasoru/resources/[epsilon]
git clone https://github.com/EpsilonAETIX/epsilon-hud.git

Manuel Indirme

  1. Ilgili scriptin GitHub sayfasina gidin
  2. Code - Download ZIP butonuna tiklayin
  3. ZIP dosyasini resources/[epsilon] klasorune cikarin
  4. Klasor adindaki -main veya -master ekini silin

Veritabani Kurulumu

Her scriptin kendi klasorunde bir sql dosyasi bulunur. Bu dosyalari veritabaniniza import edin.

Dikkat: SQL dosyalarini import etmeden once mevcut veritabaninizin yedegini almayi unutmayin.

Yapilandirma

Her scriptin kendi config.lua dosyasi vardir. Scripti baslatmadan once yapilandirmanizi tamamlayin. Detaylar icin her scriptin kendi yapilandirma sayfasina bakin.

Scripti Baslatma

server.cfg
# Bagimliliklar (once bunlar yuklenmelidir)
ensure oxmysql
ensure ox_lib
ensure es_extended  # veya qb-core

# Epsilon Scriptler
ensure epsilon-hud
ensure epsilon-inventory
ensure epsilon-garage
Sunucunuzu yeniden baslattiginizda scriptler aktif olacaktir. Herhangi bir sorunla karsilastiginizda Discord sunucumuza katilin.
