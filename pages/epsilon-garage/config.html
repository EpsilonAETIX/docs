Epsilon Garage - Configuration

Garage locations, impound settings, and capacity limits.

General Settings

config.lua
Config = {}

Config.Framework = 'esx'
Config.Language = 'tr'
Config.SaveDamage = true       -- Save vehicle damage
Config.SaveFuel = true         -- Save vehicle fuel
Config.MaxGarageCapacity = 10  -- Max vehicles per player
