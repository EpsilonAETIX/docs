Ana Sayfa Genel Ayarlar

Genel Ayarlar

Tum Epsilon scriptlerinde ortak olan genel ayarlar ve yapilandirma secenekleri.

Framework Secimi

config.lua
Config.Framework = 'esx'     -- 'esx' veya 'qbcore'

Dil Ayarlari

Scriptler coklu dil destegine sahiptir. Dil dosyalari locales/ klasorunde bulunur:

locales/tr.lua
Locales['tr'] = {
    ['inventory_full'] = 'Envanteriniz dolu!',
    ['not_enough_money'] = 'Yeterli paraniz yok!',
    ['item_used'] = '%s kullanildi.',
    ['item_received'] = '%s x%d alindi.',
}

Debug Modu

config.lua
Config.Debug = true  -- Konsola detayli log yazdirir
Dikkat: Debug modunu production sunucunuzda kapali tutun. Performans kaybina neden olabilir.

Veritabani Ayarlari

server.cfg
set mysql_connection_string "mysql://root:password@localhost/fivem?charset=utf8mb4"
