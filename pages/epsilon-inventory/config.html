Epsilon Inventory - Yapilandirma

Envanter sisteminin tum ayarlari config.lua dosyasindan yapilir.

Genel Ayarlar

config.lua 
Config = {}

Config.Framework = 'esx'
Config.Language = 'tr'

Config.MaxSlots = 50           -- Maksimum slot sayisi
Config.MaxWeight = 120000      -- Maksimum agirlik (gram)
Config.HotbarSlots = 5         -- Hotbar slot sayisi

Ozellik Ayarlari

config.lua 
Config.EnableCrafting = true    -- Crafting sistemi
Config.EnableHotbar = true     -- Hotbar sistemi
Config.EnableDrop = true       -- Yere birakma
Config.DropDespawnTime = 300   -- Yerdeki item silinme suresi (saniye)

Config.WeaponAttachments = true   -- Silah eklenti sistemi
Config.WeaponDurability = true    -- Silah dayaniklilik sistemi

AyarTipVarsayilanAciklama
