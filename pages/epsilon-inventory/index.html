Epsilon Inventory

Advanced inventory system with drag-and-drop support and modern UI. Crafting, hotbar, weapon attachments and more.

Version: 3.0.0  |  Resmon: 0.02ms  |  Framework: ESX / QBCore  |  Price: Premium

Features

  • Drag-and-drop item movement
  • Weight-based inventory system
  • Hotbar (quick access with keys 1-5)
  • Weapon attachments (silencer, scope, grip, etc.)
  • Crafting system
  • Shop, vehicle trunk, stash integration
  • Item meta data support (serial number, durability)
  • Customizable item images
  • Context menu (right click)
  • Drop items on ground (3D props)
