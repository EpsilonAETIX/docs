Genel

Hangi framework'leri destekliyorsunuz?

Tum scriptlerimiz ESX Legacy ve QBCore framework'lerini destekler. Bazi scriptler standalone modda da calisabilir.

Scriptler ucretsiz mi?

Bazi scriptlerimiz tamamen ucretsiz ve acik kaynaktir. Premium scriptler icin Discord sunucumuzu ziyaret edin.

OneSync gerekli mi?

Evet, tum scriptlerimiz OneSync (Legacy veya Infinity) gerektirir.

Sorun Giderme

Script baslatildiginda hata veriyor

F8 konsolundaki hata mesajini kontrol edin. En yaygin sorunlar:

  • Eksik bagimlilik - gerekli scriptlerin yuklenmis oldugundan emin olun
  • Yanlis framework ayari - config.lua'daki framework degerini kontrol edin
  • SQL tablolari eksik - SQL dosyalarini import ettiginizden emin olun

NUI (arayuz) acilmiyor

  • Tarayici onbellegini temizleyin: F8 konsolunda nui_devtools yazin
  • fxmanifest.lua dosyasindaki UI dosyalarinin dogru oldugunu kontrol edin

Veritabani hatasi aliyorum

  • oxmysql scriptinin yuklendiginden emin olun
  • server.cfg'deki mysql_connection_string degerini kontrol edin
  • SQL tablolarinin olusturulmus oldugundan emin olun
Cozum bulamadiysaniz Discord sunucumuza katilarak destek alabilirsiniz.
