Epsilon HUD - API / Exports

Diger scriptlerden HUD'i kontrol etmek icin kullanabileceginiz export ve event listesi.

Client Exports

HUD Gorunurluk

Client - Lua 
-- HUD'i gizle
exports['epsilon-hud']:hideHUD(true)

-- HUD'i goster
exports['epsilon-hud']:hideHUD(false)

Bar Guncelleme

Client - Lua 
-- Belirli bir barin degerini guncelle
exports['epsilon-hud']:updateBar('stress', 75)

-- Parametreler:
-- barName: 'health', 'armor', 'hunger', 'thirst', 'stress', 'oxygen'
-- value: 0-100 arasi deger

Bildirim

Client - Lua 
exports['epsilon-hud']:showNotification({
    title = 'Baslik',
    message = 'Mesaj icerigi',
    type = 'success',    -- 'success', 'error', 'warning', 'info'
    duration = 5000      -- milisaniye
})

Events

Client Events

Server - Lua 
-- HUD durumunu degistir
TriggerClientEvent('epsilon-hud:toggleHUD', source, false)

-- Ozel bar ekle
TriggerClientEvent('epsilon-hud:addCustomBar', source, {
    name = 'stamina',
    label = 'Dayaniklilik',
    color = '#bc8cff',
    value = 100
})

-- Ozel bari kaldir
TriggerClientEvent('epsilon-hud:removeCustomBar', source, 'stamina')
