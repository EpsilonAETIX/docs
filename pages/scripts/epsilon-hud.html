Ana Sayfa Scriptler Epsilon HUD

🖥️ Epsilon HUD

Tamamen özelleştirilebilir, modern ve performanslı HUD sistemi. Sağlık, zırh, açlık, susuzluk, stres, oksijen ve araç göstergeleri.

Versiyon: 2.1.0  |  Resmon: 0.01ms  |  Framework: ESX / QBCore  |  Fiyat: Ücretsiz

Özellikler

  • ✅ Sağlık, zırh, açlık, susuzluk göstergeleri
  • ✅ Stres ve oksijen barları
  • ✅ Araç hız göstergesi (km/h veya mph)
  • ✅ Araç yakıt göstergesi
  • ✅ Araç hasar göstergesi (motor, gövde)
  • ✅ Emniyet kemeri durumu
  • ✅ Minimap özelleştirme (daire/kare)
  • ✅ NUI ile kullanıcı ayarları paneli
  • ✅ Tamamen responsive tasarım
  • ✅ 0.01ms resmon değeri

Bağımlılıklar

Kaynak Zorunlu Link

Kurulum

  1. Scripti indirin ve resources/[epsilon] klasörüne atın
  2. config.lua dosyasını düzenleyin
  3. server.cfg dosyasına ekleyin:
server.cfg 
ensure epsilon-hud

Yapılandırma

Tüm ayarlar config.lua dosyasından yapılır:

config.lua 
Config = {}

-- Framework Ayarı
Config.Framework = 'esx' -- 'esx' veya 'qbcore'

-- Genel Ayarlar
Config.DefaultTheme = 'dark'     -- 'dark' veya 'light'
Config.SpeedUnit = 'kmh'         -- 'kmh' veya 'mph'
Config.MinimapShape = 'circle'   -- 'circle' veya 'square'
Config.ShowStreetName = true     -- Sokak adı gösterimi
Config.ShowCompass = true        -- Pusula gösterimi

-- Bar Ayarları
Config.Bars = {
    health = { show = true, color = '#3fb950' },
    armor  = { show = true, color = '#58a6ff' },
    hunger = { show = true, color = '#d29922' },
    thirst = { show = true, color = '#58a6ff' },
    stress = { show = true, color = '#f85149' },
    oxygen = { show = true, color = '#79c0ff' },
}

-- Araç HUD
Config.VehicleHUD = {
    showSpeed = true,
    showFuel = true,
    showDamage = true,
    showSeatbelt = true,
    fuelScript = 'LegacyFuel' -- Yakıt scripti adı
}

Yapılandırma Tablosu

Ayar Tip Varsayılan Açıklama
Config.Framework string 'esx' Kullanılacak framework
Config.SpeedUnit string 'kmh' Hız birimi
Config.MinimapShape string 'circle' Minimap şekli
Config.ShowCompass boolean true Pusula gösterimi

Exports

Diğer scriptlerden HUD'ı kontrol etmek için aşağıdaki export'ları kullanabilirsiniz:

HUD'ı Gizle/Göster

Client - Lua 
-- HUD'ı gizle
exports['epsilon-hud']:hideHUD(true)

-- HUD'ı göster
exports['epsilon-hud']:hideHUD(false)

-- Belirli bir barı güncelle
exports['epsilon-hud']:updateBar('stress', 75)

Bildirim Göster

Client - Lua 
-- Bildirim gönder
exports['epsilon-hud']:showNotification({
    title = 'Epsilon HUD',
    message = 'Hoş geldiniz!',
    type = 'success', -- 'success', 'error', 'warning', 'info'
    duration = 5000
})

Events

Client Events 
-- HUD durumunu değiştir
TriggerClientEvent('epsilon-hud:toggleHUD', source, false)

-- Özel bar ekle
TriggerClientEvent('epsilon-hud:addCustomBar', source, {
    name = 'stamina',
    label = 'Dayanıklılık',
    color = '#bc8cff',
    value = 100
})
Not: Export'ları kullanmadan önce epsilon-hud scriptinin server.cfg'de epsilon-hud'dan önce ensure edilmiş olduğundan emin olun.
