Modules

Aetix Computer consists of 12 modular components. Each module can be enabled or disabled per faction via config_factions.lua.

Dashboard

The main landing page when opening the MDT. Provides an at-a-glance overview of current activity.

  • Active dispatch calls count and recent calls list
  • Active warrants summary
  • Active BOLOs summary
  • Online units count per faction
  • Quick-access buttons to frequently used modules
  • Officer information (name, callsign, badge, faction, role)

Dispatch / CAD

Real-time Computer Aided Dispatch system for managing calls and unit assignments.

Features

  • Call Management — Create, update, and close dispatch calls with 10-codes, priority levels, and location data
  • Status Workflow — Calls progress through: PENDING → ACTIVE → CLOSED
  • Unit Assignment — Assign/unassign officers to calls. Track who is responding
  • Notes & Timeline — Add notes to calls. Timeline auto-tracks: call created, units assigned/unassigned, status changes, notes added
  • Location System — Automatic street name + area resolution via bridge system. GPS coordinates with map blips
  • Real-time Updates — All changes sync instantly across all connected MDT clients
  • 30 Dispatch Codes — Pre-configured 10-codes with priority levels and map blip IDs (configurable)
  • Map Blips — Configurable duration, color, and alpha for dispatch blips on the minimap
Dispatch calls can also be created programmatically via the bridge system. See Bridge / API for details.

Persons

Search and view detailed profiles for all citizens (players) in the database.

Features

  • Search — Search by name, state ID, or date of birth
  • Profile View — Full citizen profile with: name, DOB, gender, phone number, mugshot, physical description
  • Flags — Attach/remove person flags (Wanted, Armed & Dangerous, Gang Member, etc.). 10 built-in flags, fully customizable
  • Notes — Add officer notes to person profiles with timestamps and author tracking
  • SMT (Supervised Management Team) — Track individuals under supervision (parole, probation). Manage conditions and check-ins
  • Linked Records — View all warrants, BOLOs, citations, and reports linked to this person
  • Physical Description — Customizable via the bridge system (hair, tattoos, clothing, etc.)

Vehicles

Search and manage vehicle records with flag and ALPR integration.

Features

  • Search — Search by plate number, owner name, or vehicle model
  • Vehicle Profile — Full details: plate, model, color, class, owner information
  • Vehicle Flags — Mark vehicles as Stolen, BOLO, Wanted, or custom flags
  • Stolen Vehicle Management — Flag/clear stolen status. Stolen vehicles appear in dispatch notifications
  • Owner Lookup — Click-through to owner's person profile
  • ALPR / Radar Integration — wk_wars2x bridge automatically checks scanned plates against vehicle flags and active BOLOs. See Bridge / Radar
Vehicle MDT (in-vehicle access via F6) is only available to jobs listed in Config.VehicleMDT.allowedJobs.

Warrants

Create and manage warrants with an optional judge approval workflow.

Features

  • 5 Warrant Types — Arrest, Search, Bench, Wiretap, Seizure (configurable)
  • Judge Approval — Warrants can require COMMAND-level approval before becoming active
  • Status Workflow — PENDING → APPROVED → ACTIVE → SERVED / CANCELLED
  • Serve & Cancel — Officers can mark warrants as served or cancel them with a reason
  • Person Linked — Warrants are linked to person profiles via citizen ID
  • Search & Filter — Filter warrants by type, status, person name

BOLOs

Be On the Lookout bulletins for persons, vehicles, or general alerts.

Features

  • Create BOLOs — Title, description, person info, vehicle plate/model/color
  • Vehicle Plate BOLOs — When a BOLO includes a plate number, it integrates with the ALPR/radar system
  • Auto-expire — BOLOs can have an expiration time
  • Broadcast — New BOLOs are broadcast to all connected MDT clients in real-time
  • Active/Expired Filtering — Filter view by active or expired BOLOs

Reports

Comprehensive report writing system with 5 report types and an approval workflow.

Features

  • 5 Report Types — Incident Report, Arrest Report, Use of Force, Traffic Collision, Field Interview
  • Rich Text Editor — Full narrative editor for report body
  • Linked Calls — Link reports to dispatch calls by selecting from active/recent calls
  • Incident ID Linking — Each report can be linked to an incident ID for cross-referencing
  • Approval Workflow — Reports go through: DRAFT → SUBMITTED → APPROVED / REJECTED
  • Command Approval — Only COMMAND-level officers can approve or reject reports
  • Officer Attribution — Author, editor, and approver are all tracked

Citations

Issue citations (tickets) using the penal code system.

Features

  • Penal Code Lookup — Search 73 charges across 8 categories
  • Charge Selection — Select one or multiple charges per citation
  • Auto-calculated Fines — Total fine amount calculated from selected charges
  • Jail Time — Jail time (months) calculated from charge values
  • Person Linked — Citations are linked to the offender's person profile
  • Citation History — View all past citations for a person

Gang Intel

Gang intelligence database for tracking gang organizations and their members.

Features

  • Gang Profiles — Name, description, territory, threat level
  • Member Management — Add/remove persons as gang members with rank info
  • Activity Log — Track gang-related incidents and activity
  • Linked Persons — Cross-reference with person profiles (auto-applies gang_member flag)

Messages

Real-time internal messaging system between officers within the MDT.

Features

  • Real-time Chat — Instant messaging between online officers
  • Faction-wide — Messages are scoped to your faction
  • Message History — Persistent message history stored in database
  • Timestamps — All messages include sender name, badge, and timestamp
  • Live Updates — New messages appear instantly without refresh

Incident History

Archive of all closed dispatch calls with filtering and search capabilities.

Features

  • Call Archive — All closed calls are automatically archived
  • Search & Filter — Filter by date range, dispatch code, priority, keyword
  • Full Timeline — View the complete timeline of events for any archived call
  • Linked Reports — See all reports linked to an incident
  • Read-only Access — Historical data cannot be modified

Units

Manage on-duty units and their status for dispatch coordination.

Features

  • Unit List — View all online units across your faction
  • Status Tracking — Officers can set status: Available, Busy, En Route, On Scene, Off Duty
  • Callsign System — Predefined callsigns or dynamic callsign generation
  • Unit Management — COMMAND-level officers can manage unit assignments
  • Dispatch Integration — Units can be assigned to calls directly from the dispatch module
